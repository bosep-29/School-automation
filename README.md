# School-Automation

A School Automation system to manage students, teachers, and school activities. Implements CRUD operations for student data, attendance, and other school-related functionalities.

## Features

- Manage students, teachers, and classes
- CRUD operations for student and staff records
- Track attendance and performance
- Simple and intuitive interface for school management
